6. CONCEPTS CHÍNH
================================================================================

6.1. Basic Morphology
- Erosion (Xói mòn)
- Dilation (Giãn nở)
- Opening (Mở) = Erosion → Dilation
- Closing (Đóng) = Dilation → Erosion

6.2. Advanced Morphology
- Morphological Gradient = Dilation - Erosion
- Top-hat = Image - Opening
- Black-hat = Closing - Image
- Hit-or-Miss Transform (3 giá trị: 0, 1, -1)

6.3. Binary Morphology
- Connected Components (4-connectivity, 8-connectivity)
- Hole Filling (3 phương pháp)
- Component Properties (area, perimeter, centroid, circularity, ...)

6.4. Grayscale Morphology
- Grayscale Erosion (min operation)
- Grayscale Dilation (max operation)
- Grayscale Opening/Closing
- Grayscale Top-hat/Black-hat

6.5. Watershed Segmentation
- Marker-based Watershed
- Distance Transform
- Sure Foreground / Sure Background / Unknown
- Over-segmentation problem

6.6. Distance Transform
- 3 loại metrics: L1 (Manhattan), L2 (Euclidean), L∞ (Chessboard)
- Ứng dụng: Tìm tâm vật thể, Skeleton, Medial Axis
