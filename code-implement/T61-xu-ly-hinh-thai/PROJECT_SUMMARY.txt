6. KỸ THUẬT SỬ DỤNG
================================================================================

MORPHOLOGICAL OPERATIONS:
1. Erosion              - Thu nhỏ vật thể
2. Dilation             - Mở rộng vật thể
3. Opening              - Khử nhiễu (Erosion → Dilation)
4. Closing              - Lấp lỗ (Dilation → Erosion)
5. Gradient             - Trích biên (Dilation - Erosion)
6. Top-hat              - Vật thể sáng (Img - Opening)
7. Black-hat            - Vật thể tối (Closing - Img)
8. Hit-or-Miss          - Phát hiện pattern

ADVANCED TECHNIQUES:
- Distance Transform
- Watershed Segmentation
- Connected Components
- Contour Analysis
- Otsu Thresholding

STRUCTURING ELEMENTS:
- MORPH_RECT (Chữ nhật)
- MORPH_ELLIPSE (Elip)
- MORPH_CROSS (Chữ thập)
- Custom (Tùy chỉnh)
