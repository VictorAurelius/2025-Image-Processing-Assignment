================================================================================
KỸ THUẬT
================================================================================

MORPHOLOGICAL OPERATIONS:
1. Erosion              - cv2.erode()
2. Dilation             - cv2.dilate()
3. Opening              - cv2.morphologyEx(MORPH_OPEN)
4. Closing              - cv2.morphologyEx(MORPH_CLOSE)
5. Gradient             - cv2.morphologyEx(MORPH_GRADIENT)
6. Top-hat              - cv2.morphologyEx(MORPH_TOPHAT)
7. Black-hat            - cv2.morphologyEx(MORPH_BLACKHAT)
8. Hit-or-Miss          - cv2.morphologyEx(MORPH_HITMISS)

ADVANCED TECHNIQUES:
- Distance Transform    - cv2.distanceTransform()
- Watershed             - cv2.watershed()
- Connected Components  - cv2.connectedComponents()
- Contours              - cv2.findContours()
- Threshold Otsu        - cv2.threshold(THRESH_OTSU)

STRUCTURING ELEMENTS:
- MORPH_RECT            - Hình chữ nhật
- MORPH_ELLIPSE         - Hình elip
- MORPH_CROSS           - Hình chữ thập
- Custom numpy array    - Tùy chỉnh
